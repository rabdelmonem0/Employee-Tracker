  ## Descritpion
  This employee tracker was created using MySQL and Node.js to create a table to be able to keep track of all employees, managers, and departments.

  ## Installation
  Node modules, inquirer, and console.table are required for this app to work. In the command line run ``npm install`` to install node modules, run ``npm install inquirer`` to install inquirer, and run ``npm install console.table --save`` to install console.table

  ## Usage
  In the command line run ``node index.js`` to start the frist function to get the application started.
